# 🎨 UtilityHub Landing Page

A modern and responsive landing page inspired by SaaS platforms like **UsabilityHub**, built using **HTML5** and **CSS3**. The project showcases a clean user interface, responsive layouts, reusable CSS utility classes, and modern web design principles.

## 🌐 Live Demo

🚀 https://utility-hub-website-liard.vercel.app/

---

## 🚀 Features

- ✅ Responsive Navigation Bar
- 🏢 Trusted Companies Showcase
- ⭐ Feature Cards Section
- 📖 Product Highlight Sections
- 🖼️ Interactive Examples Showcase
- 📢 Call-to-Action Section
- 📄 Footer & Subfooter
- 🎨 Modern Typography
- 📱 Responsive Layout
- ⚡ Hover Effects & Smooth Transitions

---

## 🛠️ Technologies Used
